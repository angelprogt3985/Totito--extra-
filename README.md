# Laboratorio extra - Totito - Angel Camargo - 24003664          

Para este laboratorio, se útilizo el código del tutorial de totito en la página de react, el objetivo principal fue editar dicho archivo de tal forma en la que le dimos un mejor estilo, con más colores, más animaciones, otro tipo de letra, un mejor centrado en la pantalla. Para esto en este archivo lo único que tuvimos que modificar fue el archivo styles.css en donde manejamos todo el estilo de nuestra página web, lo que añadimos de extra para hacerlo más vistozo fue, primeramente se le cambio el color del fondo a nuestro, después de este cambio, editamos la orientación del totito, de tal manera en la que lo centramos en la pantalla para darle una mejor orientación, procedente a esto se le añadio un estilo de tarjeta blanca al rededor del totito para darle una mejor perspectiva y que al momento de que alguien este usando dicho programa se centre especificamente en el totito, procedente a esto cambiamos el tipo de letra por uno que se adaptara mejor al tipo de programa, y finalmente editamos los colores de los cuadrados del totito, por unos más llamativos que combinaran de una mejor forma con el fondo, igualmente a estos cuadrados se les añadio una animación al momento de pasar el mouse sobre ellos y al momento de darles click igual, esto para darle una experiencia mucho más dinamica y mucho más vistoza al usuario.
